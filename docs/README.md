# Documentation

Design documents, literature review, and supporting material that
will become the body of the final IEEE report.

## Planned contents

- references.bib — BibTeX entries built incrementally as I read
- literature_review.md — one paragraph per source, three buckets:
  (1) multi-agent LLM systems
  (2) AI governance for agentic systems
  (3) healthcare-specific AI governance
- test_plan.md — scenarios, pass criteria, evaluation rubric
- nist_mapping.md — table mapping prototype capabilities to NIST
  AI RMF functions (built in Phase 3)
- risk_controls.md — risk-to-control mapping (built in Phase 3)
- evaluation_rubric.md — how each scenario is judged (built in
  Phase 3)
- final_report.docx — the IEEE-format report itself (Phase 5,
  starting July 15)

## Phase plan

- Phase 1 (May 25 – Jun 1): references.bib, literature_review.md,
  test_plan.md (skeleton)
- Phase 3 (Jul 1-7): nist_mapping.md, risk_controls.md,
  evaluation_rubric.md
- Phase 5 (Jul 15-28): final_report.docx using all of the above
